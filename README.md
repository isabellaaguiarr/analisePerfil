# Análise de Perfil de Crédito

## Descrição
Este projeto tem como objetivo realizar a análise de perfil de crédito de clientes a partir de um conjunto de dados. A análise busca identificar padrões que ajudem a compreender o comportamento financeiro e o risco de crédito associado a cada perfil.

## Objetivo
- Explorar e entender os dados de crédito
- Realizar tratamento e limpeza do dataset
- Identificar características relevantes para avaliação de risco
- Apoiar a tomada de decisão em concessão de crédito

## Tecnologias Utilizadas
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn (para visualização de dados)
- Jupyter Notebook

## Etapas da Análise
1. Carregamento do dataset
2. Limpeza e tratamento dos dados
3. Análise exploratória (EDA)
4. Visualização de padrões
5. Interpretação dos resultados
